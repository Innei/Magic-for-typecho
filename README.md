### 食用方法

在主题设置里修改你的博客地址，填写Github用户名。

填写的你的博客数据库地址和密码。

- 支持Halo博客的文章解析

**( 不建议 ) 顺手写了一个文章预览页，但是只有在没有设置任何博客下启用，无评论，无独立页面。不推荐。这只是一个<span style="color: red">个人主页而不是博客。</span>**

关于友链的格式：

```
荫茵小窝
https://blog.yiny.ml
静かな森
https://shizuri.net
```

它应该是符合这种格式的。

每两行作为一个友链，第一行为标题，第二行为url。从第一行开始。
